# Ас үй рецептері сайты — recipe-app

## Жоба туралы
Бұл жоба "Ас үй рецептері сайты" веб-қосымшасы.
Node.js + Express + MongoDB технологияларымен жасалған.

## Технологиялар
- **Backend:** Node.js + Express.js
- **Дерекқор:** MongoDB + Mongoose
- **Шаблондар:** EJS
- **Аутентификация:** bcrypt + express-session
- **Валидация:** express-validator

## Орнату
```bash
git clone https://github.com/isa2408/recipe-app.git
cd recipe-app
npm install
npm run dev
```

## Орта айнымалылары (.env)
```bash
PORT=3000
MONGODB_URI=mongodb://localhost:27017/recipe-app
SESSION_SECRET=your-secret-key
```
